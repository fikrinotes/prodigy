# Prodigy — STEM Learning Hub


**Prodigy** adalah platform kurasi sumber belajar terpadu yang berfokus pada bidang STEM (Science, Technology, Engineering, and Mathematics). Platform ini menyediakan berbagai materi pembelajaran berkualitas tinggi untuk berbagai disiplin ilmu seperti Sains, Matematika, Data Science, Artificial Intelligence (AI), Programming, Web Development, dan masih banyak lagi.

Misi utama dari Prodigy adalah untuk menyediakan akses belajar gratis, terstruktur, dan terkurasi dengan baik bagi siapa saja yang ingin mendalami dunia STEM dari tingkat dasar hingga lanjutan.

## ✨ Fitur Utama

- **Koleksi Terkurasi:** Ribuan sumber belajar terbaik yang dikumpulkan dan disusun secara terstruktur.
- **Berbagai Disiplin Ilmu STEM:** Meliputi Sains, Matematika, Pemrograman, Data Science, hingga AI.
- **Akses Gratis:** Semua sumber daya yang direkomendasikan dan disusun di platform ini difokuskan pada materi yang dapat diakses secara gratis.
- **Desain Modern & Responsif:** Antarmuka pengguna (UI) yang dirancang secara premium, responsif di semua perangkat, dan memberikan pengalaman pengguna yang mulus (dibangun dengan Next.js 14+).
- **SEO & Social Media Ready:** Dilengkapi dengan metadata Open Graph (OG) penuh, memastikan tautan yang dibagikan ke WhatsApp, X, atau LinkedIn tampil dengan judul, deskripsi, dan *preview image* yang cantik.

## 🗂️ Sumber Data & Kurasi

Seluruh sumber belajar, materi, tautan, dan data resource yang tersedia di dalam platform Prodigy ini **telah dikurasi secara konsisten selama 5 tahun oleh Fikri Mulyana Setiawan**. 

Dedikasi selama setengah dekade ini ditujukan untuk memfilter materi-materi usang, mengelompokkan jalur belajar (*learning path*) yang optimal, dan menyajikan hanya konten pendidikan STEM terbaik dari seluruh penjuru internet ke dalam satu tempat.

## 🚀 Teknologi yang Digunakan

Prodigy dibangun menggunakan teknologi web modern untuk menjamin performa dan fleksibilitas:
- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Library Utama:** React, TypeScript
- **Styling:** CSS Modules dengan kustomisasi variabel yang estetik
- **Font:** Oxanium dan Outfit (Google Fonts via `next/font`)

## 🛠️ Cara Menjalankan Project Secara Lokal

Untuk menjalankan project ini di komputer Anda:

1. Clone repositori ini:
   ```bash
   git clone https://github.com/fikrinotes/prodigy.git
   ```
2. Masuk ke direktori project:
   ```bash
   cd prodigy
   ```
3. Install semua dependensi:
   ```bash
   npm install
   # atau menggunakan pnpm/yarn
   ```
4. Jalankan server development:
   ```bash
   npm run dev
   ```
5. Buka [http://localhost:3000](http://localhost:3000) di browser Anda untuk melihat hasilnya.
